# sample-python-docker
파이썬 Docker 프로젝트 예제 파일

## 1. 주요 기능
+ 도커 이미지 자동 생성 및 실행 프로젝트
+ 파이썬 가상화 pipenv 이용하여 자동 패키지 설치
+ python 3.10.1 버전으로 생성

## 2. 이력 
+ 2022.01.08 최초 생성

<br>

##  3. 사용방법
<br>

### 3.1 도커 이미지 생성
```
sh docker-init.sh
```
<br>

### 3.2 도커 이미지 시작
```
sh docker-run.sh
```
<br>

### 3.3 도커 이미지 종료
```
sh docker-stop.sh
```
